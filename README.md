A Python command-line Random Password Generator that creates strong passwords using letters, digits, and symbols.

---

## 📌 Overview

Built as **Task 3** of my **DecodeLabs Internship (Batch 2026)**.

DecodeLabs is a virtual internship program that trains students to think like real backend developers. Task 3 focuses on two core Python skills — **Importing Modules** and **String Manipulation** — to build a practical security tool used in real-world applications every day.

---

## 🚀 Features

- 🔐 Generates strong random passwords
- 🔢 User-defined password length
- 🔡 Uses letters, digits, and symbols
- ⚡ Lightweight and fast — no external libraries needed
- 🧩 Clean and beginner-friendly code structure

---

```
Enter password length: 10
Your password: aB3#kL9!mZ
```

## 🛠️ Modules Used

| Module | Purpose |
|--------|---------|
| `random` | Picks a random character each time |
| `string` | Provides ready-made character sets |

---

## 🧠 Concepts Practiced

- Importing Built-in Modules (`random`, `string`)
- String Concatenation — building password character by character
- For Loop with `range()`
- `random.choice()` for random selection
- `string.ascii_letters`, `string.digits`, `string.punctuation`
